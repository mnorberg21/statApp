
### Golf Statistics Tracking App

---

#### Dependencies:

1. Docker

#### Instructions:

1. Clone repository

```.sh
$ git clone https://github.com/mnorberg21/statApp.git
```

2. Start app with docker compose

```.sh
$ docker compose up -d
```

3. Navigate to `localhost:8080`

4. To take down application with docker compose

```.sh
$ docker compose down
```
